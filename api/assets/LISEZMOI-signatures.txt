  SIGNATURES E-MAIL — MODE D’EMPLOI
================================================================================

%%INTRO%%

--------------------------------------------------------------------------------
  CE QUE CONTIENNENT LES DOSSIERS
--------------------------------------------------------------------------------

  • EXEMPLES
    Images JPG « faux lecteur de mail » : corps de message factice + votre signature,
    pour visualiser le rendu dans un contexte proche d’un mail (aperçu généré
    automatiquement, pas une capture Gmail exacte).

  • HTML
    Les fichiers de signature au format HTML : ce sont ceux à utiliser si vous
    souhaitez des liens cliquables (site web, e-mail, téléphone, réseaux sociaux).

  • JPG et PNG
    Des images exportées de chaque signature. Pratique pour voir le design
    ou pour d’autres usages, mais pas idéal comme seule signature dans Gmail :
    voir l’encadré ci-dessous.


--------------------------------------------------------------------------------
  IMPORTANT : JPG / PNG ET LIENS CLIQUABLES
--------------------------------------------------------------------------------

  Si vous insérez votre signature uniquement comme une image (fichier JPG ou PNG),
  ce que les gens voient est une PHOTO : les numéros, adresses et icônes ne
  sont pas de vrais liens Internet. On ne peut pas cliquer dessus.

  Pour avoir une signature avec des liens qui fonctionnent dans Gmail, il faut
  passer par le fichier HTML (méthode expliquée plus bas).


--------------------------------------------------------------------------------
  MÉTHODE RECOMMANDÉE : HTML → SITE « PROBADOR » → GMAIL
--------------------------------------------------------------------------------

  1) Ouvrez la version HTML que vous préférez
     Dossier « HTML », par exemple :
       nomclient_signature-v3.html
     avec un éditeur de texte simple, par exemple le Bloc-notes (Notepad) sous
     Windows. Vous verrez tout le code : c’est normal.

  2) Tout sélectionner et copier
     Raccourci : Ctrl+A pour tout sélectionner, puis Ctrl+C pour copier.
     (Vous pouvez aussi sélectionner à la souris et utiliser le menu contextuel :
     Clic droit → Copier.)

  3) Aller sur le site qui prépare la signature pour Gmail
     Ouvrez ce lien dans votre navigateur :
       https://solamentecodigoshtmlbybcn.jimdofree.com/probador-de-html/

     Collez votre code dans la zone prévue (premier champ / zone de texte du
     probador), avec Ctrl+V ou clic droit → Coller.

     Lancez l’action prévue par le site (bouton du type « convertir », « voir
     résultat », etc. — le site ouvre en général une NOUVELLE fenêtre ou un
     nouvel onglet avec l’aperçu de votre signature).

  4) Copier le résultat affiché dans le navigateur
     Sur cette nouvelle page : Ctrl+A pour tout sélectionner, puis Ctrl+C pour
     copier (ou clic droit → Copier).

  5) Coller dans Gmail
     Dans Gmail : Paramètres (roue dentée) → « Voir tous les paramètres »
     → onglet « Général » → section « Signature ».

     Créez une nouvelle signature (bouton « Créer »), donnez-lui un nom, placez le
     curseur dans la grande zone de texte, puis Ctrl+V (ou clic droit →
     Coller).

     Enregistrez en bas de page (« Enregistrer les modifications »).

  Vous devriez ainsi obtenir la mise en page souhaitée avec des liens cliquables.


--------------------------------------------------------------------------------
  MODIFIER UN TEXTE DE VOTRE SIGNATURE DANS GMAIL
--------------------------------------------------------------------------------

  Une fois la signature collée dans Gmail, vous pouvez la modifier directement,
  sans repasser par le fichier HTML. Par exemple pour changer le prénom/nom,
  l'intitulé de poste, ou corriger une faute.

  1) Ouvrez l'éditeur de signature
     Dans Gmail : Paramètres (roue dentée en haut à droite) → « Voir tous les
     paramètres » → premier onglet « Général » → descendez jusqu'à la section
     « Signature ». Votre signature s'y affiche dans une zone d'édition.

  2) Modifiez le texte directement
     Cliquez dans la signature comme dans un traitement de texte :
       • Pour SUPPRIMER : sélectionnez le texte et appuyez sur Suppr / Retour
         arrière.
       • Pour AJOUTER ou CORRIGER : placez le curseur à l'endroit voulu et tapez.
       • Pour METTRE EN FORME : sélectionnez le texte, puis utilisez la barre
         d'outils en bas de la zone (Gras B, Italique I, Souligné, couleur,
         taille de police, etc.).

  3) Enregistrez
     Descendez en bas de la page et cliquez sur « Enregistrer les modifications ».

  Astuce : modifiez uniquement le texte. Évitez de supprimer le logo ou les
  éléments de mise en page si vous souhaitez conserver le design d'origine.
